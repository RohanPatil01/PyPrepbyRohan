String Methods

⚡ Key Points
Strings are immutable: all methods return new strings.
Indexing is zero-based. Negative indices count from the end.
s[::-1] reverses a string. s[a:b:step] is the slice syntax.
s.upper() / s.lower()
Convert case. Returns new string.
'hello'.upper() → 'HELLO'
s.strip(chars)
Remove leading/trailing whitespace (or chars). lstrip/rstrip for one side.
' hi '.strip() → 'hi'
s.split(sep, maxsplit)
Split string into list. Default splits on whitespace.
'a,b,c'.split(',') → ['a','b','c']
sep.join(iterable)
Join iterable into string using sep. Opposite of split.
','.join(['a','b','c']) → 'a,b,c'
s.replace(old, new, count)
Replace all (or count) occurrences of old with new.
'aaa'.replace('a','b',2) → 'bba'
s.find(sub) / s.index(sub)
find returns -1 if not found. index raises ValueError.
'hello'.find('ll') → 2
s.startswith(prefix) / s.endswith(suffix)
Check start/end of string. Returns bool.
'hello'.startswith('he') → True
s.count(sub)
Count non-overlapping occurrences of substring.
'banana'.count('a') → 3
s.isdigit() / s.isalpha() / s.isalnum()
Check character types. All return bool.
'123'.isdigit() → True
s.format(*args) / f'{expr}'
String interpolation. f-strings preferred (Python 3.6+).
f'Hello {name!r}' / f'{pi:.2f}'
s.zfill(width)
Pad with zeros on the left to reach width.
'42'.zfill(5) → '00042'
s.title() / s.capitalize() / s.swapcase()
Case transformations. title() capitalises each word.
'hello world'.title() → 'Hello World'
# Slicing quick reference s = 'Python' s[0] # 'P' - first char s[-1] # 'n' - last char s[1:4] # 'yth' - index 1,2,3 s[::-1] # 'nohtyP' - reversed s[2::2] # 'to' - every 2nd from index 2

List Methods

⚡ Key Points
Lists are mutable: methods like append/sort modify in-place and return None.
sorted() returns a new list; list.sort() sorts in-place.
Assignment (b=a) creates an alias, not a copy. Use a[:] or list(a) for shallow copy.
lst.append(x)
Add single element to end. O(1) amortized.
[1,2].append(3) → [1,2,3]
lst.extend(iterable)
Add all elements from iterable to end.
[1,2].extend([3,4]) → [1,2,3,4]
lst.insert(i, x)
Insert x before index i. O(n).
[1,3].insert(1,2) → [1,2,3]
lst.remove(x)
Remove first occurrence of x. Raises ValueError if absent.
[1,2,2].remove(2) → [1,2]
lst.pop(i=-1)
Remove and return element at index i. Default: last.
[1,2,3].pop() → 3, list=[1,2]
lst.index(x)
Return index of first occurrence of x.
[10,20,30].index(20) → 1
lst.count(x)
Count occurrences of x in list.
[1,1,2].count(1) → 2
lst.sort(key, reverse)
Sort in-place. Returns None. Use sorted() for new list.
lst.sort(key=lambda x: -x)
lst.reverse()
Reverse in-place. Returns None. lst[::-1] for new list.
[1,2,3].reverse() → [3,2,1]
lst.copy()
Shallow copy. Same as lst[:] or list(lst).
b = a.copy() # independent list
lst.clear()
Remove all elements. Same as del lst[:].
[1,2,3].clear() → []
len(lst) / sum(lst) / min/max
Built-in functions that work on lists.
sum([1,2,3]) → 6

Dictionary Methods

⚡ Key Points
Keys must be hashable (str, int, tuple: not list/dict).
Dicts are ordered by insertion order since Python 3.7.
Use get() instead of [] to avoid KeyError on missing keys.
d.get(key, default=None)
Return value or default. Never raises KeyError.
d.get('x', 0) → 0 if 'x' missing
d.keys() / d.values() / d.items()
View objects (live, reflect changes). Wrap in list() if needed.
for k, v in d.items(): ...
d.update(other)
Merge other into d. Existing keys overwritten.
d.update({'a':1, 'b':2})
d.pop(key, default)
Remove and return value. Raises KeyError if no default.
d.pop('x', None) → safe removal
d.setdefault(key, default)
Return value if key exists; else insert and return default.
d.setdefault('x',[]).append(1)
d.copy()
Shallow copy of dictionary.
d2 = d.copy()
d1 | d2 (Python 3.9+)
Merge dicts into new dict. d2 wins on conflicts.
merged = {**d1, **d2} # older syntax
d.popitem()
Remove and return last inserted (key, value) pair.
k, v = d.popitem()
# Common dict patterns # Frequency count freq = {} for x in items: freq[x] = freq.get(x, 0) + 1 # Invert a dict inv = {v: k for k, v in d.items()} # Group by key from collections import defaultdict groups = defaultdict(list) for item in data: groups[item.key].append(item)

Set Operations

s.add(x)
Add element. No effect if already present.
s.remove(x) / s.discard(x)
remove raises KeyError; discard doesn't.
s.union(t) or s | t
All elements in either set.
{1,2}|{2,3} → {1,2,3}
s.intersection(t) or s & t
Elements in BOTH sets.
{1,2}&{2,3} → {2}
s.difference(t) or s - t
In s but NOT in t.
{1,2,3}-{2,3} → {1}
s.symmetric_difference(t) or s ^ t
In either but NOT both.
{1,2}^{2,3} → {1,3}
s.issubset(t) / s.issuperset(t)
Check containment relationships.
s.isdisjoint(t)
True if no common elements.

Important Built-in Functions

len(x)
Number of items in a sequence/collection.
range(start, stop, step)
Lazy range object. range(5) → 0,1,2,3,4.
enumerate(iterable, start=0)
Yields (index, value) pairs.
for i, v in enumerate(['a','b']): ...
zip(*iterables)
Pair up elements. Stops at shortest.
zip([1,2],['a','b']) → (1,'a'),(2,'b')
map(fn, iterable)
Apply fn to each element. Returns iterator.
list(map(str, [1,2,3])) → ['1','2','3']
filter(fn, iterable)
Keep elements where fn returns True.
list(filter(None, [0,1,2,False])) → [1,2]
sorted(iterable, key, reverse)
Return new sorted list. Works on any iterable.
sorted(d, key=d.get, reverse=True)
any(iterable) / all(iterable)
any: True if any truthy. all: True if all truthy.
any([0,0,1]) → True / all([1,1,0]) → False
isinstance(obj, type)
Check type with inheritance. Preferred over type().
isinstance(True, int) → True
type(obj)
Return exact type. No inheritance consideration.
type([]) → <class 'list'>
id(obj)
Memory address of object. Used with 'is' operator.
hash(obj)
Hash value (only for hashable/immutable objects).
repr(obj) / str(obj)
repr: developer string. str: user-friendly string.
abs / round / divmod / pow
Math built-ins. divmod(a,b) → (quotient, remainder).
divmod(17, 5) → (3, 2)

Comprehensions

# List Comprehension [expr for item in iterable if condition] squares = [x**2 for x in range(10) if x % 2 == 0] # Dict Comprehension {k: v for k, v in iterable if condition} inv = {v: k for k, v in d.items()} # Set Comprehension {expr for item in iterable} unique_lens = {len(w) for w in words} # Generator Expression (lazy - no brackets) total = sum(x**2 for x in range(1000000)) # Nested Comprehension (flatten) flat = [x for row in matrix for x in row]

# Lambda - anonymous single-expression function square = lambda x: x**2 add = lambda x, y: x + y clamp = lambda x, lo, hi: max(lo, min(hi, x)) # Common uses sorted(people, key=lambda p: p['age']) sorted(words, key=lambda w: (len(w), w)) # multi-key max(items, key=lambda x: x.score) # map / filter list(map(lambda x: x**2, nums)) list(filter(lambda x: x > 0, nums)) # functools from functools import reduce, lru_cache, partial reduce(lambda a, b: a + b, [1,2,3,4]) # → 10 @lru_cache(maxsize=None) # memoisation decorator def fib(n): return n if n <= 1 else fib(n-1) + fib(n-2) double = partial(pow, exp=2) # partial application
